# Images for Dreamlife Marketing Site

Please add the following images to this folder. The site will gracefully handle missing images by hiding them.

## Required Images:

1. **hero-dream-plan-live.png** (or .jpg)
   - Desktop screenshot showing "Dream. Plan it. Live it." with coffee shop background
   - Recommended size: 1200x800px or larger
   - Used in: Hero section

2. **lifestyle-simulation.png** (or .jpg)
   - Screenshot showing "Living the Dream in Santa Barbara, California" with house image and expenses overview
   - Recommended size: 800x600px or larger
   - Used in: About section and Screenshots section

3. **desktop-dream-dashboard.png** (or .jpg)
   - Desktop view of the dashboard showing "Dream. Plan it. Live it."
   - Recommended size: 1200x800px or larger
   - Used in: Screenshots section

4. **mobile-dream-dashboard.png** (or .jpg)
   - Mobile view showing "Dream it. Plan it. Live it." with beach condo visualization
   - Recommended size: 400x800px (mobile aspect ratio)
   - Used in: Features section and Screenshots section

5. **retirement-projection.png** (or .jpg)
   - Retirement projection screen with graphs showing portfolio growth
   - Recommended size: 800x600px or larger
   - Used in: Features section and Screenshots section

6. **app-menu.png** (or .jpg)
   - Menu/navigation screen showing all app features (Dashboard, Simulations, Plan & Money, etc.)
   - Recommended size: 400x800px (mobile aspect ratio)
   - Used in: Features section

## Image Format:
- PNG or JPG format
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Keep file sizes reasonable (under 500KB each if possible)

## Note:
The HTML includes `onerror` handlers that will hide images if they're not found, so the site will still work without images, just without the visual showcase.
